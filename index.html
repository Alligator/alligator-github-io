Games Done Quick Tracker

2013 onwards

I've been running an unofficial viewers/donations tracker for the biannual Games Done Quick marathons since 2013.

The UI is written in JavaScript, using d3 for charts. The data fetching parts are written in Python.

talkbox

2019

A discord bot, designed to be simple to understand and easy to extend.

It's based around plugins that add commands to the bot.Plugins can be added to and removed from the bot at run-time and it's pretty fault tolerant so a bad plugin won't take the whole bot down. This is to make adding new plugins easy and fast.

Written in JavaScript
