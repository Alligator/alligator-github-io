AGDQ & SGDQ Stats

AGDQ (Awesome Games Done Quick) and SGDQ (Summer Games Done Quick) are speedrunning marathons held in the winter and the summer respectively. I made a simple page to track the number of viewers over time for SGDQ in 2013 and I've been iterating on it for each marathon since.

SGDQ 2014 Stats

This is a small iteration on the previous version. A new colour scheme and some minor optimizations mean it runs faster and has less gaps.

SGDQ 2014 Stats

AGDQ 2014 Stats

The was the first version to use d3.js. I made it look all nice and shiny with a new-ish layout and a colour palette from Adobe Kuler. The backend was still quite wobbly so the graph has lots of gaps and strange data points.
