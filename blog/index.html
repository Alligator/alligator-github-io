Mar 8, 2021

A friend showed me an interesting feature of this language, comments that remove the next syntactic element.

I thought this was a cool idea so I implemented it in my toy lisp. #- comments out the next element, so multi-line function definitions can be removed with one comment:

#- (def add-two (x)
  (+ x 1))

or single list items can be removed inline:

(+ 1 #- 2 3)
becomes
(+ 1 3)

More langauges need this thing.
